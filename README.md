\## Etapas del Pipeline



| Etapa | Descripción |

|---|---|

| Build | Compilación del proyecto |

| Test | Ejecución de pruebas automatizadas |

| SCA | Análisis de dependencias vulnerables |

| SAST | Análisis estático de código |

| Security Gate | Criterios de aceptación de seguridad |



\## Descripción



Pipeline diseñado con etapas de seguridad integradas siguiendo prácticas

DevSecOps: escaneo de dependencias, análisis estático y gates de seguridad

antes del despliegue.
